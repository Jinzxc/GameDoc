# GameDoc
# Frontend Setup
Go to frontend directory: 
```
cd frontend/game-doc
```

## 1. Install Dependencies
You will need to install `Node.js`, `npm` to manage JavaScript packages that we are going to use.

**Download & Install Node.js & npm**  
https://nodejs.org/en/

**Install node modules in package.json**  
Make sure you are in the **`frontend/game-doc` directory** before you do this.
```
npm install
```
## 2. Happy hacking!
You can run the app in the development mode by using
```
npm start
```
Open http://localhost:3000 to view it in the browser.  

To build the app for production to the build folder.
```
npm build
```

# Backend Setup
## 1. Install Dependencies
- Install Python
## 2. Setting Up the Virtual Environment
1. Create virtual environment (Python3)
 - For Linux/Mac: python3 -m venv env
 - For Windows: py -m venv env
3. Set up a new virtual environment
 - For Linux/Mac: source venv/bin/activate
 - For Windows: \venv\bin\activate.bat

4. Install required libraries
    > pip install -r requirements.txt 
## 3. Running The APIs
1. > python app.py
2. Open http://localhost:5000
